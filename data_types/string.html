Строки

В языке Java Script, на момент написания курса, выделяют следующие типы данных:

  • Главная страница
  • number - числа
  • null - null, или отсутствие значения
  • undefined - пустое значение
  • boolean - булевые значения
  • object - объект
  • function - функция
  • symbol - специальный символ

Строки представляют собой последовательность символов, ограниченных двойными или одинарными кавычками. При этом, если нужно внутри строки выделить текст кавычками,
то нужно использовать кавычки, отличные от тех, которыми определяется строка.

Пример: let firstString = "This is string type." Ещё пример: let secondString = "This is 'string type' to."
